О программе "Сапёр"

Данная программа, являющаяся клоном логической игры "Сапёр", представляет собой портфолио программиста на Java и не предназначена для тиражирования.
Целью разработки была демонстрация навыков построения программных продуктов, а также оформления и документирования исходного кода.
Рисунки и идея игрового дизайна взяты из версии игры для Windows XP.
